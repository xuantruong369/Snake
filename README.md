# 🐍 SnakeAI – LibGDX Game Project

## 📌 Giới thiệu

SnakeAI là project game Snake được xây dựng bằng **LibGDX + Java**, hỗ trợ chạy đa nền tảng (Desktop, Android…).
Project được thiết kế để làm việc nhóm, dễ mở rộng AI, gameplay và UI.

## 🧱 Công nghệ sử dụng

- Java 17
- LibGDX
- Gradle
- Git & GitHub

## 📂 Cấu trúc thư mục chính

SnakeAI/

- core/ Logic game chính
- assets/ Ảnh, âm thanh, font
- lwjgl3/ Desktop launcher
- android/ Android launcher

## 🚀 Chạy game (Desktop)

```bash
./gradlew lwjgl3:run
```

## 👥 Quy ước làm việc nhóm

- Không code trực tiếp trên main
- Làm việc qua dev / feature branch
- Pull trước khi push

## 🌿 Git Workflow

main ← dev ← feature/\*
